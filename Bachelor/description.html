Erklärung

Eine Dezimalzahl kannst du ganz einfach in eine Binärzahl umwandeln, indem du die Zahl durchgehend durch 2 dividierst und so einen Rest erhälst. Wenn das Ergebnis 0 beträgt, bist du fertig und bekommst eine Binärzahl.
